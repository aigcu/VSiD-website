Our Mission

The VSID (Very Small Image Detection) system will be installed in a hallway on a college/university campus. It will consist of a camera connected to a microcontroller with onboard Wi-Fi. A TinyML model will be trained to detect thumbs up or thumbs down gestures from the video stream. Once trained, the model will be packaged and prepared for deployment on the microcontroller. The microcontroller will continuously run the model and monitor the video stream. The results of the model, collected over a set time period, will be sent to a remote database via Wi-Fi. Data analysis will be performed on the results, which will be queried from the database and displayed in a website format.
